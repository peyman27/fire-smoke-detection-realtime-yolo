# Realtime Fire & Smoke Detection

This project provides a simple real-time fire and smoke detection system using a trained PyTorch model.

# Project Structure

├─ assets
├─ core
│  ├─ camera.py
│  ├─ models.py
│  └─ utils.py
├─ main.py
├─ models
│  └─ best.pt
├─ README.md
└─ requirements.txt
